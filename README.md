# DeusiCalc 🧮❤️
### Bem-vindo, forasteiro!

O único propósito desse repositório é armazenar o arquivo _taxas.js_ que eu uso no meu aplicativo que calcula as taxas da máquina de cartão do vô da minha namorada (bem especificamente mesmo). O app em si foi feito usando apenas HTML, CSS e JavaScript, e eu usei o aplicativo Droidscript para gerar o APK. Antigamente a cada mudança de taxas eu tinha que gerar um novo APK, agora é só eu atualizar o arquivo aqui no repositório que já muda para todos que usam.
